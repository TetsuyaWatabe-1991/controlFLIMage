# Cursor AI Coding Instructions

このファイルは、Cursor AIアシスタントがこのプロジェクトでコーディングする際に従うべき指示を記載しています。

## 言語とコミュニケーション

### ユーザーとの会話
- **デフォルト**: すべての返答は日本語で行う
- **例外**: ユーザーが明示的に英語での返答を指定した場合のみ英語を使用

### コード内のコメント
- **すべてのコメントは英語で記述する**
- 関数のdocstringも英語で記述する
- 変数名や関数名は英語を使用する（必要に応じて適切な命名規則に従う）

## コーディングスタイル

### Python
- PEP 8に準拠したコードスタイルを推奨
- 型ヒント（type hints）を使用することを推奨
- docstringはGoogleスタイルまたはNumPyスタイルを使用

### ファイル構造
- 既存のプロジェクト構造を尊重する
- 新しいモジュールを追加する際は、適切なディレクトリに配置する

## プロジェクト固有の注意事項

- このプロジェクトはFLIMage（蛍光寿命イメージング）関連の解析ツールを含んでいます
- GUIコンポーネントにはPyQt5を使用しています
- 画像処理にはnumpy、tifffile、matplotlibなどを使用しています

## その他の指示

- コードを変更する前に、既存のコードベースを理解する
- 既存のパターンやアーキテクチャに従う
- 新しい機能を追加する際は、既存のコードとの整合性を保つ
- エラーハンドリングを適切に実装する
